Tutorial menjalankan kode utama
(Pastikan semua data di folder tidak hilang)

Step 1: Direkomendasikan membuka folder Code di VsCode
Step 2: Download python di perangkat masing-masing
Step 3: Buka terminal VsCode di folder ini
Step 4: Jalankan
        pip install streamlit pandas joblib scikit-learn
                    atau
        py -m pip install streamlit pandas joblib scikit-learn

Step 5: Jalankan
        Streamlit run ModelAi.py
                atau
        py -m streamlit run ModelAi.py
